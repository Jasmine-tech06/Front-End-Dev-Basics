# Employee Management System

A React-based Employee Management System that performs CRUD operations using MockAPI.

## Features

- Welcome Page
- Dashboard
- Employee Directory
- Add Employee
- Update Employee
- Delete Employee
- Search Employee
- Statistics Dashboard

## Technologies Used

- React
- JavaScript
- CSS
- MockAPI
- Axios

## Testing

The project contains detailed software testing documentation inside the `Testing` folder.
